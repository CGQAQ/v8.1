# HOW TO BUILD

## Prerequisites
- Git
- Python3

## Build (Non-Windows)
```bash
git clone --recurse-submodules https://github.com/CGQAQ/v8.1.git v8.1
cd v8.1
python3 build-v8.py
```

## Build (Windows)
```bash
git clone --recurse-submodules https://github.com/CGQAQ/v8.1.git v8.1
cd v8.1
python build-v8.py
```
